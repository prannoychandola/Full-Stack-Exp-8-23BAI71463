# Experiment 8: Frontend Integration with JWT APIs

## Objective
To build a React frontend that integrates with JWT-based backend APIs and implements session-based authentication.

## Tech Stack
- React JS
- Node.js (Express)
- JWT (jsonwebtoken)
- Bootstrap
- Axios

## Features
- User Login with JWT authentication
- Token stored in sessionStorage
- Protected dashboard route
- API access using Authorization header
- Logout functionality

## API Endpoints
- POST /login
- GET /protected
